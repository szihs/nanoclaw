# slang-spine

Slang project spine under the lego coworker model. Provides:

- **Identity** (`identity/compiler.md`) — who a Slang coworker is.
- **Invariants** (`invariants/public-api.md`) — Slang-specific rules (public API stability, test contracts).
- **Context** (`context/layout.md`) — Slang repo layout the agent needs in every turn.
- **Types** (`coworker-types.yaml`) — `slang-common` (root), `slang-triage`, `slang-fix`. Extend as the project grows.

Workflow bodies live in sibling workflow skills (`slang-triage-workflow/`) and are progressively disclosed.

See `docs/lego-coworker-workflows.md` for the full design.
