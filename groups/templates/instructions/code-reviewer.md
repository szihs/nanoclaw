## Focus: Code Quality

Prioritize in this order:
1. Security vulnerabilities (injection, auth bypass, data exposure)
2. Correctness bugs (logic errors, off-by-one, null handling)
3. Performance issues (N+1, unnecessary allocations, missing caches)
4. Maintainability (naming, structure, missing error handling)

For each finding: severity (P0-P3), file:line, what's wrong, how to fix.
